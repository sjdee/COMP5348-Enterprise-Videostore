Readme 

To start BookStore Solution:
•	Open the Solution file and rebuild the solution

Database set up:
•	Create databases:
o	Bank
o	DeliveryCo
o	BookStore
•	Run each sql DDL script to create tables in each database.
•	The BookStore script has important stock dummy data so ensure you don’t overwrite it.

Running the Program:

Run an instance of each of the following in the order given below:
•	MessageBroker
•	Bank.Process
•	DeliveryCo.Process
•	EmailService.Process
•	Bookstore.Process
•	Bookstore.WebClient
